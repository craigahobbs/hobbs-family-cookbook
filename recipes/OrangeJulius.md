# Orange Julius

~~~ markdown-script
recipeHeader(objectNew( \
    'author', 'Teresa Ball' \
))
~~~

Place all ingredients in a blender, cover and blend until smooth.

~~~ markdown-script
recipeIngredients(arrayNew( \
    '6 oz orange juice concentrate', \
    '1 C milk', \
    '1 C water', \
    '1/4 C sugar', \
    '1/2 tsp vanilla extract', \
    '12 ice cubes' \
))
~~~

Serve immediately.
